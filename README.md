Problem Statement

A customer database system is required to store and manage customer records efficiently.
The system should support operations such as insertion, deletion, searching, and finding the position of a specific customer using a singly linked list.

 Data Structures Used
 Singly Linked List
 Algorithms Used
 Linear Search
 Traversal Algorithm
 Features
 Add customer records (beginning & end)
 Insert records at a specific position
Delete customer records
 Search for a customer
 Find position of a specific element
 Display all records
 Tech Stack
 Java (Core Java)
